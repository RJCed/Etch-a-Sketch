# Etch-a-Sketch

A browser-based drawing application inspired by the classic Etch-A-Sketch toy. Users can draw by hovering over a grid and customize the drawing experience with different grid sizes and color modes.

## Live Demo

Preview: https://rjced.github.io/Etch-a-Sketch/

## Features

* Draw by hovering over grid cells
* Adjustable grid size
* Reset/Clear grid button
* Random color mode
* Shading effect for darker drawings
* Responsive browser-based interface
* (Will make Modals for Grid and Strenght button in the future)
